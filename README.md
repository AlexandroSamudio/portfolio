## 🚦 Cómo Ejecutar Localmente

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/AlexandroSamudio/portfolio.git
   cd portfolio
   ```

2. **Instalar dependencias**
   ```bash
   npm install
   ```

3. **Ejecutar**
   ```bash
   ng serve
   ```

4. **Abrir en el navegador**
   ```
   http://localhost:3000
   ```
